# 🧠 Natural Language to SQL Chatbot

A FastAPI-powered chatbot that translates natural language into SQL queries using OpenAI and runs them on a SQLite database.

## 🚀 Features

- 🗣️ Converts user text like “List all students” into SQL
- 🤖 Uses OpenAI's GPT-3.5 to generate SQL
- 🧾 Executes on a SQLite database with a `students` table
- ⚡ FastAPI backend + interactive Swagger UI
